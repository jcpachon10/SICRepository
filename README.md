## What is SIC ?
- Software generates  packing  patterns  from  user-defined instances of the Container Loading Problem for regular and irregular cargo
- Provides convenient tool to generate, edit, save, export and delete containers loading plans.

## Dependencies:
_Unity_, _IBM ILOG CPLEX Optimization Studio 20.10 in its academic version_, _LeanTween_ 

## Install
### Install executable
Downloads  the ['SIC_Executable'](https://github.com/jcpachon10/SICRepository/SIC_Executable) folder where you going to find  an executable file. Click on it and you will be able to use the softare. It s necessary to install ['IBM ILOG CPLEX Optimization Studio 20.10 for Windows x86-64 '](https://www-03.ibm.com/isc/esd/dswdown/searchPartNumber.wss?partNumber=CJ8GXML) for irregular shape module.

### Install Unity Proyect
Downloads  the ['SIC_Executable'](https://github.com/jcpachon10/SICRepository/SIC_UnityProject) folder where you going to find  all the files  of the the developer version. In Unity with version 019.4.20f1 or higher and you need to install ['LeanTween'](https://assetstore.unity.com/packages/tools/animation/leantween-3595#content) from the Asset Store in Unity becouse it´s a necesary  for the UI of SIC.
